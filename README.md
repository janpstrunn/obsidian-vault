## My personal Obsidian Vault as a Template

This is my personal Obsidian Vault containing a collection of decent amount of snippets and plugins configured to my own liking and workflow.

Besides all the configuration, the vault has several note templates as well, ranging from the most basic notes with proprieties to dataview examples.
